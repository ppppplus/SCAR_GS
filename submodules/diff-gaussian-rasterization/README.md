# mcgs_gaussian_rasterization

**mcgs_gaussian_rasterization** is a CUDA-based differentiable rasterizer for **multi-channel 3D Gaussian Splatting**.  
It is adapted from [graphdeco-inria/diff-gaussian-rasterization](https://github.com/graphdeco-inria/diff-gaussian-rasterization), with significant extensions to support **multi-channel rendering** and **gradient backpropagation** for additional feature modalities such as **semantic labels** and **affordances**.
